By Quentin 'WatchYourNachos' Dupupet (in association with DeepHouseShika Records)

  • git push -f is the recommended way of pushing files to ensure your amazing changes are recognized.
  • git status checks the status of a repo.
  • git add tells git a file or changes are important.
  • git commit tells git to save a set of changes with a message.
  • git pull tells git to pull completed commits from a server.
  • git push tells git to send completed commits to a server.
  • git implode tells git to BURN IT ALL TO THE GROUND.
  • git genesis tells git to create a new religion and wage war under its auspices.
